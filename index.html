Perfil profesional

Soy desarrollador de software con experiencia en backend, frontend y sistemas ERP. Creo soluciones robustas, eficientes y escalables, adaptadas a las necesidades del negocio. Mis herramientas principales incluyen Python (Flask y Django), Java EE, PHP (Symfony) y Node.js, trabajando con bases de datos como Oracle Database, PostgreSQL y SQL Server.

Me especializo en integración de sistemas, procesos ETL y optimización de procesos, con sólidas habilidades de comunicación, resolución de problemas y adaptabilidad.

También tengo conocimientos de diseño web y manejo herramientas como Figma, Adobe XD, Photoshop, Illustrator, Premiere Pro, InDesign, CapCut y Canva, además de Microsoft Office a nivel avanzado.
